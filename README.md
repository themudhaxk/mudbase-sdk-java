# MUDBASE JAVA SDK

Auto-generated SDK for MUDBASE API.

## Installation

```xml
<dependency>
  <groupId>dev.mudbase</groupId>
  <artifactId>mudbase-sdk</artifactId>
  <version>1.0.0</version>
</dependency>
```
