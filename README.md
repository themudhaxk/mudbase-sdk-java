# MUDBASE java SDK

Official **java** client for the [MUDBASE](https://mudbase.dev) platform.  
Packages are versioned from our OpenAPI spec so they stay in sync with the API.

## Installation

```xml
<dependency>
  <groupId>dev.mudbase</groupId>
  <artifactId>mudbase-sdk</artifactId>
  <version>1.0.0</version>
</dependency>
```

## Usage

See [API documentation](https://docs.mudbase.dev/api) and generated source for endpoint-level examples.
